Serviço de computação de alta vazão para suporte a atividades de e-ciência, executado sobre uma federação de nuvens.

Visão geral

O Iguassu é um serviço para monitoramento e execução de tarefas de e-ciência que exigem alta vazão. Os recursos utilizados para execução destas tarefas fazem parte de uma federação de nuvens, gerenciadas pelo middleware Fogbow (fogbowcloud.org). O Iguasse permite que seus usuários usem os recursos das cloud sem se preocupar com detalhes da infraestrutura.
